This archive contains **expanded, stitch-perfect** patterns with:
- **Size tables** (Standard 10–13 cm + Keychain mini)
- **Gauge math** (how to scale stitches/rounds from your gauge to target size)
- Step-by-step instructions in **tables per section**
- Standardized **image/diagram prompts** for automation (Typist-ready friendly)

## Conventions
- R = round, Row = row
- MR = magic ring
- sc = single crochet
- inc = 2 sc in same stitch
- dec = invisible decrease (sc2tog)
- sl st = slip stitch
- ch = chain
- ( … ) xN = repeat N times
- Counts shown as **(total sts)**

## Recommended materials (baseline)
- Yarn: DK cotton (≈ 220–250 m/100 g)
- Hook: 2.5 mm (Standard), 2.0 mm (Keychain)
- Stuffing: polyester fiberfill
- Needle, stitch markers, scissors
